# 🚀 Hero IO — App Marketplace

## 📌 Description

Hero IO is a modern and responsive web application that showcases a collection of mobile applications with detailed insights. Users can explore trending apps, search through all available apps, view detailed information including ratings and downloads, and install their favorite apps.

The application also includes a personalized "My Installation" feature where users can manage installed apps, along with sorting and filtering functionalities for better user experience.

---

## ✨ Features

* 🔍 Live Search (case-insensitive)
* 📱 View All Apps with details
* ⭐ Ratings & Reviews Chart (Recharts)
* 📥 Install App (saved in localStorage)
* 📦 My Installation Page
* ❌ Uninstall Apps
* ⚡ Loading Animation
* 🔔 Toast Notifications
* 📱 Fully Responsive Design

---

## 🛠️ Technologies Used

* React.js
* React Router
* Tailwind CSS
* Recharts
* React Icons
* React Toastify
* LocalStorage (Browser API)
